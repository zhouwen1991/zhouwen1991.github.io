一 安装Hexo

1 安装git
　git教程参考

2 安装nodejs

3 安装Hexo

　打开git bash，输入下面代码
　$ npm install hexo-cli -g
　创建一个网站目录blog,然后通过blog目录创建站点
　$ mkdir blog
　$ hexo init blog
　进入blog站点目录，安装npm插件
　1 $ cd blog
　2 $ npm install

二 安装主题模板

　1 在Hexo主题页面，进入blog目录，把模板导入themes文件
　$ git clone https://github.com/tufu9441/maupassant-hexo.git themes/maupassant
　2 安装主题所需插件，github页面会有详细说明，列如：

　3 配置blog目录下的_config.yml文件,将theme的值改为主题文件名，如maupassant，还有一些其他配置如下：

　

三 生成博客文件，并在本地预览

　1 生成public静态文件
　$ hexo g
　2 启动服务器
　$ hexo s
　3 在浏览器中查看http://localhost:4000/，关闭服务器按ctrl+c

四 上传到github和gitcafe

　1 注册平台账号，用户名为服务器地址前缀，github是国外的平台，gitcafe是国内的
　2 创建新仓库，仓库名必须为用户名加网站特定的后缀，如：username/github.io ，并勾选公开和初始化项目，其他默认即可
　3 生成本机public key ，添加id_rsa.pub文件里的公匙到平台
　ssh-keygen -t rsa -b 4096 -C "xxx@xxx.com"
　4 多平台配置，参考如下格式：

5 安装hexo-deploy-git插件：在blog目录下执行
　$ npm install hexo-deployer-git --save
　6 最后进行部署,第一次通过公钥上传会弹出提示框，输入yes，再点ok，没有提示表示上传失败，说明公钥配置有问题
　$ hexo deploy
