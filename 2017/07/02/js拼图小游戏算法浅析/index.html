一 实现原理及算法

　　任何程序，只要掌握了原理和算法，实现它不过是代码熟练度的问题。算法才是核心！
　　js拼图的原理：利用css背景图片的position坐标规律，js改变css属性值，即可实现拼图碎片的移动。实际上div并没有移动，只是改变了div的背景图坐标而已。
　　js拼图的算法：以九宫格为列，模拟碎片移动到空白区域实现操作，根据观察可知可以移动的碎片和空白区域的关系为上下左右相邻，根据这一规律可以得到一个位置关系，用算法表示出来作为事件判断的依据即可
